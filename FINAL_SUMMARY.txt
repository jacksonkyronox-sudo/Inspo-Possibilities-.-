================================================================================
                           TROUBLESHOOTING
================================================================================

Problem: "ModuleNotFoundError: No module named 'google'"
Fix: pip install -r requirements.txt

Problem: "Authentication failed" 
Fix: Check your API key in .env (no spaces, correct key)
     Get new key at: https://aistudio.google.com/app/apikey

Problem: "Connection refused"
Fix: Make sure python app.py is running

Problem: "Port 5000 already in use"
Fix: Edit app.py line 71, change port=5000 to port=5001

Problem: "Is this really free?"
Answer: YES! 100% free forever. No hidden charges.
